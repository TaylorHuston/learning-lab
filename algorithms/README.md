# Algorithms & Data Structures

Classic algorithm and data structure implementations, consolidated from multiple repos (2014-2015).

## Directory Structure

```
algorithms/
├── sorting/              # Sorting algorithms
│   ├── java/             # QuickSort, MergeSort, HeapSort, etc.
│   └── javascript/       # JS implementations
│
├── linked-lists/         # Linked list implementations
│   ├── java/             # Singly/doubly linked, ArrayList
│   └── javascript/       # JS linked list
│
├── stacks-queues/        # Stack and queue implementations
│   └── java/             # ArrayStack, PriorityQueue, BinaryHeap
│
├── graphs/               # Graph algorithms
│   └── java/             # BFS, DFS, Topological sort, SCC
│
├── trees/                # Tree data structures
│   └── java/             # BST, Symbol tables
│
├── shortest-path/        # Path-finding algorithms
│   └── java/             # Dijkstra, Kruskal MST, Prim MST
│
└── interview-prep/       # Interview problem solutions
    └── cracking-the-coding-interview/
```

## Sources

- **Algorithms, 4th Edition** by Sedgewick & Wayne - Most Java implementations
- **Cracking the Coding Interview** - Interview prep problems
- **Eloquent JavaScript** - JS implementations

## Notes

These were originally separate repos created while learning. The implementations use helper classes (StdIn, StdOut, etc.) from the Algorithms 4th Edition course materials.
